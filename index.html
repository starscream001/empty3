UI-тесты Effective Mobile

README в формате лендинга

Одностраничное описание проекта UI-тестов главной страницы effective-mobile.ru с инструкциями по запуску локально и в Docker.

Склонировать репозиторий Docker — руководство
Что проверяем:
  • Навигацию по шапке: «О нас», «Вакансии», «Отзывы», «Контакты».
  • Ссылки и поведение блока «Услуги».
  • Переходы к якорям #services и #contact с проверкой отображения блоков.

Тесты используют Playwright + Pytest в связке с паттерном Page Object и могут запускаться локально или в контейнере.

Быстрый старт локально

  1. Создайте и активируйте виртуальное окружение: 
    python3 -m venv .venv
source .venv/bin/activate
  2. Установите зависимости и скачайте Chromium для Playwright: 
    pip install -r requirements.txt
playwright install --with-deps chromium
  3. Запустите тесты с сохранением артефактов Allure: 
    pytest tests/ui --alluredir=allure-results
  4. По желанию откройте отчёт Allure: 
    allure serve allure-results

Если PyPI недоступен, проверьте прокси: без playwright и pytest-playwright тесты будут помечены как пропущенные.

Запуск в Docker

Собрать образ

docker build -t effective-mobile-tests .

Выполнить тесты

docker run --rm -v $(pwd)/allure-results:/app/allure-results \
  effective-mobile-tests

По умолчанию внутри контейнера запускается pytest tests/ui --alluredir=allure-results.

Клонирование и структура

Клонировать репозиторий

git clone <repo-url>
cd <repo-folder>

После клонирования следуйте инструкциям выше для локального запуска или Docker.

Структура проекта

  • tests/ui/pages/ — Page Object (базовый класс и главная страница).
  • tests/ui/test_home_navigation.py — тесты навигации.
  • tests/ui/conftest.py — фикстуры Pytest, интеграция Playwright.
  • Dockerfile — окружение для запуска тестов.
  • requirements.txt — зависимости проекта.

Технологии и подход

Проект написан под Python 3.10+ и использует стек для надёжной автоматизации UI:

  • Playwright
  • Pytest + pytest-playwright
  • Allure для отчётности
  • Page Object pattern
  • Docker для изолированного запуска

Тесты проверяют доступность целевых блоков сайта и корректность навигации. При проблемах с сетью убедитесь, что домен www.effective-mobile.ru доступен из вашего окружения.
