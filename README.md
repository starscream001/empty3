# Гирудотерапия — одностраничный сайт

Одностраничный сайт-визитка для услуги «Гирудотерапия». Сделан на чистом HTML/CSS/JS, полностью адаптивный.

## Как открыть локально

1. Скачайте проект или клонируйте репозиторий.
2. Откройте файл `index.html` двойным кликом в браузере.

Дополнительно можно запустить простой локальный сервер:

```bash
python3 -m http.server 8000
```

После этого сайт будет доступен по адресу `http://localhost:8000`.

## Настройка отправки формы в Telegram

По умолчанию форма работает в режиме fallback: показывает сообщение «Спасибо!» и предлагает позвонить.

Чтобы включить отправку в Telegram:

1. Откройте файл `script.js`.
2. Заполните переменные:
   ```js
   const TELEGRAM_BOT_TOKEN = "ваш_токен";
   const TELEGRAM_CHAT_ID = "ваш_chat_id";
   ```
3. Сохраните файл и обновите страницу.

## Деплой на GitHub Pages

1. Загрузите проект в GitHub-репозиторий.
2. Откройте вкладку **Settings → Pages**.
3. В разделе **Build and deployment** выберите:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (или ваша ветка), `/root`
4. Сохраните. Через минуту GitHub покажет ссылку на опубликованный сайт.
